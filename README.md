# Time & Flow

A clean, minimalist Pomodoro timer built with vanilla JavaScript.

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling (13KB)
- `app.js` - Application logic (25KB)

## Features

- **Work/Break Cycles**: 25min work → 5min short break → 15min long break (every 4 sessions)
- **Customizable Durations**: Adjust timer lengths in settings
- **Persistent State**: Timer state saved to cookies (optional)
- **Sound Notifications**: Pleasant chimes when timers complete
- **Wake Lock**: Prevents screen from sleeping during timer
- **Fullscreen Mode**: Press `F` or double-tap (mobile)
- **Keyboard Shortcuts**:
  - `Space` - Play/Pause
  - `R` - Reset (running) / Switch mode (paused)
  - `F` - Toggle fullscreen
